# Project Overview

CollabFlow is a realtime collaboration platform similar to Trello.

# Tech Stack

Frontend:
- React JS
- MUI
- Zustand
- TanStack Query
- DnD Kit

Backend:
- Node.js
- Express.js
- PostgreSQL
- Prisma ORM
- Socket.IO
- JWT Authentication
- Zod Validation
- Redis

# Coding Standards

- Keep components modular
- Use service layer abstraction
- No business logic in controllers
- Use DTOs
- Follow clean architecture
- Reusable UI components
- Keep websocket logic isolated

# Features

- JWT authentication
- Workspace management
- Boards
- Columns
- Tasks
- Realtime task updates
- Drag and drop
- Activity logs
